# Social Movements and Organized Struggle Dataset on Turkish Twitter/X

This repository provides the annotation documentation for the **Social Movements and Organized Struggle (SOMOS) Dataset on Turkish Twitter/X**.

The SOMOS dataset is a computational inventory of social movement organizations active on Turkish Twitter/X. The annotation framework documents how accounts were screened and how organizational type, inequality axis, ideology, movement scope, and location were coded.

## Annotation manual

The annotation manual is available in three formats:

- [Read the manual on GitHub](SOMOS_Annotation_Manual.md)

## Dataset and citation

The archived dataset is available through Harvard Dataverse:

> Kına, Fuat; Akdeniz, Reyta; Ekmen, Helin Yaren; Cebe, Ali; Çukur, Ayşenur; Avran, Rabia; Kosova, Ayşe Merve; Gündüz, Hüsnü Berk; Karataş, Mehmet Ali; Türkoğlu, Alp Ata. 2026. “Social Movements and Organized Struggle Dataset on Turkish Twitter/X.” Harvard Dataverse, V2. https://doi.org/10.7910/DVN/PL1DC5

## Repository scope

This repository currently hosts the annotation manual and citation metadata associated with the dataset. The DOI-based archival record in Harvard Dataverse is the authoritative source for the released dataset and preserved versions.
